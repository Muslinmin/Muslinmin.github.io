Project Objective

Develop a maze code/algorithm to solve the maze without pre-mapping

System Showcase

Simulation of code in Gazebo

Functionalities

Recursive Depth first search with SLAM and ROS2

  • Recurses every level of four options; Up, Down, Left, Right
  • Recurses upwards to look at other options if meets dead end

Skills used

  • ROS2
  • Python
  • SLAM
