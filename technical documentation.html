Key challenges and resource constraints faced

  • Two weeks deadline (10 business days)
  • Less than SGD $400 budget
  • Deliverables that need to be done:
    • Pixy cam guided X axis movement and printing routine
    • Pixy cam guided Y axis conveyer movement
    • Full integration of X and Y axis movement and printing routine

Re-used existing materials from previous project

  • The conveyer belt, timing pulleys and extrusion bars were from previous project
  • Front view and side view showcasing how E.A.S.Y. X axis is where printpen travels, Y axis is the conveyer belt
  • 3D printed Push button mechanism to press the Print pen once finished

The embedded side; Electronics and Software:

  • Code written in C
  • Utilize STM32 F103RB board and its peripherals
  • The circuit isolates the signal portion from the high voltage, high current load
×

The algorithm

  • The code takes a snapshot upon detection of pastry
  • Create a row of pastry objects to be printed along the x axis
  • Using the lowest centre-point among objects in the row, to push the entire row towards underneath the print carriage
  • To find out more about the code in detail (software documentation), please visit this GitHub repository.
